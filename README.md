# 🔐 Restraint Protocol Frontend

Restraint Protocol Frontend is a mobile-first **Ionic React application** for a Focus-to-Earn and Discipline Staking system. It connects to a Node.js backend API to help users build discipline through focus challenges, app/website restrictions, and staking-based accountability.

Built for Android (Capacitor), hackathon demos, and future Web3 expansion.

---

## 🚀 Overview

Users can:
- Create focus challenges
- Stake value on discipline goals
- Block distracting apps and websites
- Track streaks and discipline scores
- Earn rewards for completing challenges
- Get penalized for breaking commitments

---

## 🧠 Core Features

- 📱 Mobile-first UI (Ionic React)
- 🔐 JWT Authentication
- 🎯 Challenge creation & tracking
- 💰 Stake & reward system (API-driven)
- ⛔ Restriction control center
- 📊 Discipline score & streak tracking
- 👥 Accountability partner integration
- 🔔 Violation & status updates

---

## 🏗️ Tech Stack

- React (Ionic Framework)
